Gallery Vault

A simple and elegant Gallery Vault Web Application built using Python (Django) that allows users to securely store and manage their personal memories (images) in one place.

🚀 Features

🔐 User Authentication (Login & Signup)
📤 Upload images with title
🖼️ View uploaded memories in a gallery format
🎨 Clean and responsive UI
🔓 Logout functionality

🛠️ Tech Stack

Backend: Python, Django
Frontend: HTML, CSS, JavaScript
Database: SQLite (default Django DB)
Version Control: Git & GitHub
